# 手帐素材 Sheets

每个可用素材包都是一个独立 sheet。编辑器按 sheet 浏览素材，而不是把来自不同网站的图片混在同一个列表里。

## 目录约定

```text
sheets/
  <sheet-id>/
    sheet.json
    <source-image>.png
```

`sheet.json` 是唯一的素材索引，必须记录：来源页、条款链接、使用边界、默认尺寸、标签和每张图片的原始链接。

## 使用边界

- `studio-private`：只作为本地 Studio 的内置素材，不做单独下载入口。
- `public-page-use`：可出现在用户发布的手帐成品页。
- 不把第三方原始文件或修改后的素材文件重新打包、出售或提供下载。
- 每个来源单独建 sheet；不要把不同授权来源混装进一个 sheet。
